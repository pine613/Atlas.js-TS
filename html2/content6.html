Map

Let's use the Mapping function of Atlas.js
By using the mapping function,you can spread the tiles
in the canvas.
using the method called "setMap" of sprites in the initScene,
you have to call the method called "drawMapGraph" in the mainScene.

例


I'm using the image at "First Seed Material"
Suppose that you load the image width 16*16px devided by 30(vertical)*16(horizontal) like this
