Next,I'd like to introduce you the Initialization, Loading and Mainloop. 
game.initScene(fn);
/*
it runs before the loadings.
*/
game.loadingScene(fn);
/*
it runs during the loadings.
*/
game.mainScene(fn);
/*
it runs after the loadings
this method acts as gameloop.
*/
in the game.initScene, you can instance the sprite like this
Suppose that you load the image width 32*32px devided by 3(vertical)*4(horizontal)
I download the material at http://usui.moo.jp/rpg_tukuru.html 

game.load("./images/chara2.png");
var chara = new Sprite("./images/chara2.png",32,32);

you can also instance the sprite by naming the assets like this

game.load("./images/chara2.png");
game.naming(["./images/chara2.png","chara2"]);
var chara = new Sprite("chara2",32,32);
