First

Please download Atlas.js by clicking the beneath button.
DownLoad
After Setting Atlas.js to the same hierarchy with your HTML files
you can import Atlas.js as below. 
<html>
<head>
<script type="text/javascript" src="Atlas.js"></script>
</head>
<body>
</body>
</html>
Next,you have to include the canvas element with its id to the body element as following. 
<body>
<canvas id="Atlas"></canvas>
</body>
Well,if you want to know about Canvas element,please visit the following URL. http://www.html5.jp/canvas/index.html
Finaly,Writing down the JavaScript codes in the body element,
You can make your own games
<canvas id="Atlas"></canvas>
<script>
<!--You can write the code here-->
</script>
</body>
And、Suppose that the id of the canvas element is "Atlas"
Then 
var game = new Atlas('Atlas');
you have to substitute "Atlas" to the game constractor. By doing this,you can use the variety of methods provided by Atlas.js thorough the variable "game".
