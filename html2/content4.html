Sprite

var sprite = new Tile("./images/chara2.png",32,32);
if you want to duplicate above sprite, you can do like this,
var sprite2 = new Tile(sprite,32,32);
A sprite initialized like this has following properties as initial value. 
sprite.x = 0;//x point
sprite.y = 0;//y point
sprite.rot = 0;//theta
sprite.sx = 1;//enlargement ratio of x
sprite.sy = 1;//enlargement ratio of y
sprite.frame = 0;//a number of frame.

see the image on the right
Relation between the images and frames
If you set the sprite frame as 3,"3" will be drawn.

And you can use following method via sprites. 
sprite.drawGraph();//draw the sprite without maginification
sprite.drawScaleGraph();//draw the sprite with magnification
/*
In any way,the sprite will be drawn at the point of (sprite.x,sprite.y)
Please call this drawing methods in the loadingScene or mainScene;
*/
sprite.within(anotherSprite,range);
/*
Parameters are another sprite and the range
if another sprite is within the radius of the sprite's range,
return true,else return false.
*/
sprite.intersect(x,y);
/*
if the point of (x,y) is in the sprite,return true,else return false.
*/

Group Function

you can manipulate all of shape objects and tile objects via Group()

For your reference,Group is a child Class of Array

//generate group object
var enemies = new Group();
//add object to group
var enemy = new Enemy();
enemies.add(enemy);
//manipulate all added object
enemies.move(x,y);
/*
in this moment,enemies_group execute each method "enemies[i].move(x,y)"
of each added object in order. 
*/
enemies[i].remove = true;
//remove the number i object
