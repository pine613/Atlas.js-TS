各種設定

ゲームをループさせる前の初期設定のgameのメソッドを紹介したいと思います 
var game = new App();

game.load("hero.png","bgm.mp3");
/*
ゲームで使用する素材をロードします
    */
game.naming(["hero.png","hero"],["bgm.mp3","bgm"]);
/*
ゲームで使用する素材に名前を付けます
    */
game.centerize();
/*
ゲーム画面をブラウザ画面の中心に配置します。
*/
game.changeQuality(width,height);
/*
ゲーム画面の縦横の解像度を設定します。
*/
game.changeSize(width,height);
/*
ゲーム画面の縦横のサイズを設定します。
*/
game.keyEnable();
/*
ユーザーのキー操作を有効にします
    */
game.keySet(buttonA,buttonB);
/*
ボタンのキーをそれぞれ、
game.key.aにbuttonA
game.key.bにbuttonB
と、バインドさせる
キーが押されているとき、
game.key.a == true
キーが離されているとき、
game.key.a == false
*/
game.fps = 30;
/*
ゲームのfpsを30に設定します。
*/
