各種設定

ゲームをループさせる前の初期設定のgameのメソッドを紹介したいと思います 
game.centerize();
/*
ゲーム画面をブラウザ画面の中心に配置します。
*/
game.changeQuality(width,height);
/*
ゲーム画面の縦横の解像度を設定します。
*/
game.changeSize(width,height);
/*
ゲーム画面の縦横のサイズを設定します。
*/
game.keySet(buttonA,buttonB);
/*
ボタンのキーをそれぞれ、
game.key.aにbuttonA
game.key.bにbuttonB
と、バインドさせる
キーが押されているとき、
game.key.a == 1
キーが離されているとき、
game.key.a == 0
*/
game.fps = 30;
/*
ゲームのfpsを30に設定します。
*/
